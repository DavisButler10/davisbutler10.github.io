About Me


Davis Butler

Games, Animation, and Simulation | Embry-Riddle Aeronautical University

Aspiring game developer. All of these projects were made using Unity 2019/2020. This page is an accumulation of my last semesters in Game Engine Architecture, Artificial Intelligence, and Multiplayer Games Systems. Much of the code is based off the implementations from these books: Artificial Intelligence for Games by Ian Millington, Game Programming Patterns by Robert Nystrom, and Hands-On Game Development Patterns with Unity 2019 by David Baron, as well as multiplayer tutorials from Zenza Academy.
